# Research

Purpose: Store research papers that inform ideation and implementation.

File naming:
<research-topic>-<yyyy-mm-dd>.md
Example: event-sourcing-2026-01-22.md

Paper structure (academic style):
1. Basic Information
2. Research Objective / Problem Statement
3. Background and Literature Review
4. Methodology
5. Key Findings / Results
6. Discussion and Interpretation
7. Conclusion
8. Citations and References
9. Critical Evaluation

Rules:
- Cite sources with URLs.
- Keep summaries concise and actionable.
