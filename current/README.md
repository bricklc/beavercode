# Current

Purpose: Single source of truth for active work and handoffs.

Files:
- README.md (this file)
- status.md (current phase, focus, blockers)
- todo.md (prioritized task list)
- context.md (session notes and decisions)

Rules:
- Read current/ first when starting a session.
- Update status.md after significant changes.
- Update todo.md when priorities shift.
- Update context.md before ending a session.
- Do not delete history; mark items done.
